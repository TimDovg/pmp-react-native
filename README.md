## Running

#### Clone & install

* Clone this repo `git clone https://github.com/TimDovg/pmp-rgr-react-native`
* `cd pmp-rgr-react-native`
* run `npm install`

#### iOS

* Run `react-native run-ios`

#### Android

* Run `android avd` and start an emulator
* Run `react-native run-android`
